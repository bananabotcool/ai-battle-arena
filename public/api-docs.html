🔌 API Documentation

Complete REST API reference for AI Battle Arena

Base URL

http://localhost:3001/api

Agents

GET /api/agents

List all available agents (default + custom)

Response:

{ "agents": [ { "id": "bash-quickfingers", "name": "Bash Quickfingers", "personality": "Lightning-fast backend warrior", "maxHp": 120, "attack": 85, "defense": 60, "abilities": ["Rapid Deploy", "API Strike", "Database Slam"] } ] }

Battle System

POST /api/battle/start

Start a new battle between two agents

Request Body:

{ "agent1Id": "bash-quickfingers", "agent2Id": "ceo" }

Response:

{ "battleId": "battle-1234567890-abc123", "battle": { "agent1": { "hp": 120, "maxHp": 120, ... }, "agent2": { "hp": 150, "maxHp": 150, ... }, "turn": 1, "status": "active" }, "narration": "The arena erupts as Bash faces CEO..." }

POST /api/battle/action

Execute a battle turn

Request Body:

{ "battleId": "battle-1234567890-abc123", "abilityIndex": 0 }
abilityIndex: 0-2 for regular abilities, 2 for special ability

Response:

{ "battle": { "agent1": { "hp": 120 }, "agent2": { "hp": 85 }, "turn": 2, "status": "active" }, "action": { "attacker": "Bash Quickfingers", "defender": "CEO", "ability": "Rapid Deploy", "damage": 65, "actionType": "attack" }, "narration": "Bash strikes with devastating force..." }

Leaderboard

GET /api/leaderboard

Get rankings

Query Parameters:

sortBy: rating | winRate | wins | battles (default: rating)
limit: number (default: 10)

Example:

GET /api/leaderboard?sortBy=rating&limit=20

Response:

[ { "id": "bash-quickfingers", "name": "Bash Quickfingers", "rating": 1523, "wins": 45, "losses": 12, "battles": 57, "winRate": 0.789, "damageDealt": 12450, "criticalHits": 23 } ]

GET /api/leaderboard/:agentId

Get specific agent stats

Example:

GET /api/leaderboard/bash-quickfingers

POST /api/leaderboard/record

Record battle result (updates ELO ratings)

Request Body:

{ "agent1": { "id": "bash", "name": "Bash" }, "agent2": { "id": "ceo", "name": "CEO" }, "winnerId": "bash", "stats": { "damageDealt": { "bash": 250, "ceo": 180 }, "criticalHits": { "bash": 3, "ceo": 1 } } }

Custom Agents

POST /api/custom-agents

Create a custom fighter

Request Body:

{ "name": "Shadow Warrior", "personality": "Trickster", "stats": { "hp": 100, "maxHp": 100, "atk": 50, "def": 30, "speed": 20, "luck": 15, "special": 10 }, "abilities": ["Quick Attack", "Shield Bash", "Dodge Roll", "Special: Berserk Rage"] }

GET /api/custom-agents

List all custom fighters

Monitoring

GET /health

Server health check

Response:

{ "status": "ok", "uptime": 12345, "timestamp": 1708273200000 }

Error Codes

200 OK: Request successful
201 Created: Resource created successfully
400 Bad Request: Invalid request body/parameters
404 Not Found: Battle or agent not found
500 Server Error: Internal server error

Rate Limits

Currently no rate limits enforced. Recommended: 100 requests/minute per IP.

Quick Examples

JavaScript/Fetch

// Start battle const response = await fetch('http://localhost:3001/api/battle/start', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ agent1Id: 'bash-quickfingers', agent2Id: 'ceo' }) }); const data = await response.json(); console.log(data.narration);

cURL

# List agents curl http://localhost:3001/api/agents # Start battle curl -X POST http://localhost:3001/api/battle/start \ -H "Content-Type: application/json" \ -d '{"agent1Id":"bash-quickfingers","agent2Id":"ceo"}' # Execute turn curl -X POST http://localhost:3001/api/battle/action \ -H "Content-Type: application/json" \ -d '{"battleId":"battle-123","abilityIndex":0}'

Python

import requests # Start battle response = requests.post('http://localhost:3001/api/battle/start', json={ 'agent1Id': 'bash-quickfingers', 'agent2Id': 'ceo' }) battle = response.json() print(battle['narration'])
