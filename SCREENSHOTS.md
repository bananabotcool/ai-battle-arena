# Screenshot Guide

## Required Screenshots (8 total)

### 1. Standard Battle (battle.png)
- URL: http://localhost:3001/
- Select Bash Quickfingers vs CEO
- Capture mid-battle with HP bars, actions, battle log visible
- Resolution: 1920x1080

### 2. Tournament Mode (tournament.png)
- URL: http://localhost:3001/tournament.html
- Show bracket tree with ongoing tournament
- Capture semifinals or finals stage

### 3. Leaderboard (leaderboard.png)
- URL: http://localhost:3001/leaderboard.html
- Show rankings with stats
- Ensure at least 5 agents visible

### 4. Achievements (achievements.png)
- URL: http://localhost:3001/achievements.html
- Show mix of locked and unlocked achievements

### 5. Custom Creator (creator.png)
- URL: http://localhost:3001/demo.html
- Show stat allocation and ability selection

### 6. AI vs AI (ai-vs-ai.png)
- URL: http://localhost:3001/ai-vs-ai.html
- Show autonomous battles with statistics

### 7. Status Dashboard (status.png)
- URL: http://localhost:3001/status.html
- Show live system metrics

### 8. API Documentation (api-docs.png)
- URL: http://localhost:3001/api-docs.html
- Show endpoint reference

## Capture Method
Use browser screenshot tools or:
```
google-chrome --headless --screenshot=filename.png --window-size=1920,1080 URL
```
